## Quick Cherry-Pick Guide

To apply a single feature group, cherry-pick the commits in order:

```bash
# Example: Apply only the Investor Room (Feature 4)
git cherry-pick e1b81f29 105f6bb8 333b62aa e22f7a8a 9e6933a0 41c6100e 11e232c0 c947b0d4 72ee899d f8ba3357 4aedc9a8 b42c9282 1f5458b5

# Example: Apply only Friday Copilot core (Feature 1)
git cherry-pick 793bcc36 5248adbc 70c1921c dc3c97ec be9ef56f 064f898c 6e2282c4 1e27ab16 fa30b073 d3ebf48e
```

### Feature Dependencies
- **Feature 2** (Voice) depends on **Feature 1** (Copilot core)
- **Feature 3** (Checklist) depends on **Feature 1** (Copilot core)
- **Feature 5** (Nav links) depends on **Feature 4** (Investor Room)
- **Feature 6** (PDF Download) depends on **Feature 4** (Investor Room)
- **Feature 7** (Email PDF) depends on **Feature 4** and **Feature 6**
- **Features 1–3** and **Features 4–7** are independent of each other
